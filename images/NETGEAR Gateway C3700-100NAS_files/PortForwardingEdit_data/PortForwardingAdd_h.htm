Ports - Custom Services Help

To set up an application, game, or service:

  1. Type the service name in the Service Name field.
  2. Type the beginning port number in the External Starting Port field.
  3. If the application uses only a single port, type the same port number in the External Ending Port field.
  4. If the application uses a range of ports, type the ending port number of the range in the External Ending Port field.
  5. The Use the same port range for Internal port check box is selected by default. If you want to use different internal port, you can clear the check box and type the number of the internal port to which you want to redirect traffic.
  6. Type the IP address of the computer in the Internal IP Address field.
  7. Click Apply.

To edit a service entry:

  1. Modify the information in the Service Name, External Starting Port, External Ending Port, Internal Starting Port, Internal Ending Port and Internal IP Address fields.
  2. Click Apply.
